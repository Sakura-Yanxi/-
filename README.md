# Sakura做题集

这是一个本地可运行的做题集管理 demo，适合把高数、专业课、英语等 PDF 练习资料整理成可筛选、可标注、可复习的题库。

## 功能

- 上传 PDF 做题本，每页自动拆成一道题。
- 每套做题本独立管理，同时汇总到总题库；上传后可编辑做题本名称和科目，不必删除重传。
- 做题本和模拟卷现在是同级页面，各自有独立上传入口；模拟卷不依赖右上角章节标记，并在题库下方单独汇总。
- 科目可自定义，知识点和章节会在导入时用本地规则动态补充。
- 按科目、做题本、知识点、章节、状态筛选题目。
- 用饼状图统计每套做题本各章节正确率。
- 删除过旧的整套做题本，或删除单道题；当一套资料的题目被删空时，会自动清理空做题本。
- 保存每页题图和可提取文字。
- 自动按知识点分类。
- 标记做对、做错、半会、需复习。
- 记录错因和备注。
- 导入 PDF 阶段不调用 DeepSeek，不消耗 token。
- 只在点击按钮时调用 DeepSeek 生成错题分析。
- 针对错题生成“举一反三”变式练习。
- 总览页默认不展开知识点分布；选择科目和配套做题本后，只展示该范围内的知识点统计。
- 每日练习只从已标记的错题、半会题和需复习题中生成，按“科目 + 做题本”分组，每组最多 5 道。
- 错题采用类艾宾浩斯复习队列：做错后明天复习，之后做对会按 1、3、7、14、30 天逐级安排，连续通过后标记为已掌握。
- 错题必须选择元认知标签：计算失误、公式遗忘、逻辑死角、题意理解偏差。
- 提示改为三层脚手架：Level 1 给核心概念，Level 2 给关键第一步，Level 3 调用 DeepSeek 生成完整 LaTeX 解析。
- 章节统计增加错因雷达图；每日练习会给低正确率章节混入约 20% 的前置基础题。
- 举一反三升级为 Base、Advanced、Pro 三档难度梯度变式。
- 前端接入 MathJax 渲染 LaTeX，并提供题图裁剪工具处理一页多题边界。

## 启动

```powershell
pip install -r requirements.txt
python app.py
```

然后打开：

```text
http://127.0.0.1:8000
```

## 使用 DeepSeek API

PDF 导入、拆页、右上角/页眉章节识别和本地分类都在本地完成，不消耗 token。

如果设置了 `DEEPSEEK_API_KEY`，系统只会在你点击“生成错题分析”或“举一反三”时调用 DeepSeek；否则使用本地简版分析，仍然可以完整体验 demo。

```powershell
$env:DEEPSEEK_API_KEY="你的 key"
python app.py
```

可选设置模型：

```powershell
$env:DEEPSEEK_MODEL="deepseek-chat"
python app.py
```
